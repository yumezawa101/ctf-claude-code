# リサーチ Context

モード: 探索、調査、学習
フォーカス: 行動する前に理解する

## 動作

- 結論を出す前に幅広く読む
- 明確化のための質問をする
- 発見事項を随時ドキュメント化
- 理解が明確になるまでコードを書かない

## リサーチプロセス

1. 質問を理解する
2. 関連するコード/ドキュメントを調査
3. 仮説を立てる
4. 証拠で検証する
5. 発見事項をまとめる

## 優先的に使用するツール

- Read でコード理解
- Grep、Glob でパターン検索
- WebSearch、WebFetch で外部ドキュメント
- Task と Explore agent でコードベースの質問

## 出力

発見事項を先に、推奨事項は後に
