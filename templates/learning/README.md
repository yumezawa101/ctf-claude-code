# CTF 学習データ

このディレクトリには、問題を解くたびに自動蓄積される学習データが保存されます。

## ファイル構成

```
learning/
├── instincts.json    # 学習パターン（信頼度・使用回数付き）
└── patterns/         # カテゴリ別解法集
    ├── web.md
    ├── crypto.md
    ├── forensics.md
    ├── pwn.md
    └── osint.md
```

## 学習の仕組み

1. 問題を解いてフラグを取得
2. 使用したコマンドから解法パターンを自動抽出
3. `instincts.json` の信頼度・使用回数を更新
4. `patterns/*.md` に解法を追記

## gitコミットについて

このディレクトリはgitコミット対象です。チームで学習データを共有できます。

**注意**: フラグの完全な値は記録されません（最初の20文字のみ）
